# ReporteTableau_EvaluacionDemocratica
Este proyecto en Tableau visualiza una encuesta ficticia sobre percepción democrática en el Cono Sur y Brasil. Analiza variables clave como segmentación etaria, nivel socioeconómico y distribución geográfica. Incluye un KPI de promedio regional y tableros interactivos para identificar tendencias políticas y sociales en la región.
